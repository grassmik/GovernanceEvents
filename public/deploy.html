Governance Center

How to deploy on the IBM Infosphere Information Server side

The Governance Center web application displays events statistics gathered by a server-side application to be deployed on the server where IBM Infosphere INformation Server is running. To deploy the server-side application, follow the instructions below.
  1. Download jar file GC-Events-Sample-1.0-all.jar. This may take some minutes.
  2. Copy the jar file to the server where IBM Infosphere Information Server is running, and change to the directory you copied into.
  3. Execute following command:
    java -jar GC-Events-Sample-1.0-all.jar localhost demo
The last parameter "demo" produces demo events, for example in the case that the Infosphere Information Server does not produce events. It can be omitted.





Light-weight alternative

The jar file above is self-contained. Instead, you can download a smaller jar file and use the jar files already installed on the server side. Follow the instructions below.
  1. Download jar files GC-Events-Sample-1.0-all.jar and javax.json-1.0.4.jar.
  2. Copy the jar file to the server where IBM Infosphere Information Server is running, and change to the directory you copied into.
  3. Execute following command, replacing {ISInstallDirectory}by the directory where Information Server is installed:
    java -cp "javax.json-1.0.4.jar:GC-Events-Sample-1.0.jar:{ISInstallDirectory}/shared-open-source/kafka/install/libs/*" com.ibm.iis.shared_open_source.gc_events_sample.ConsumerGroupExample localhost demo
