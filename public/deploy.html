Governance Center

How to deploy on the IBM Infosphere Information Server side

The Governance Center web application displays events statistics gathered by a server-side application to be deployed on the server where IBM Infosphere INformation Server is running. To deploy the server-side application, follow the instructions below:
  1. Download jar file GC-Events-Sample.jar. This may take some minutes.
  2. Copy the jar file to the server where IBM Infosphere Information Server is running.
  3. Execute following command: java -jar GC-Events-Sample.jar localhost demo
The last parameter "demo" produces demo events, for example in the case that the Infosphere Information Server does not produce events. It can be omitted.


The jar file above is self-contained. Instead, you can download the smaller jar file GC-Events-Sample-1.0.jar. Make sure that following dependencies are available in your classpath before you call the jar file:
  • {IS Install Directory}/shared-open-source/kafka/install/libs/kafka_2.10-0.8.2.1.jar
  • {IS Install Directory}/shared-open-source/kafka/install/libs/kafka-clients-0.8.2.1.jar
  • {IS Install Directory}/shared-open-source/kafka/install/libs/scala-library-2.10.4.jar
  • javax.json-1.0.4.jar to be downloaded and stored on the server.
