How to deploy on the IBM Infosphere Information Server side

The GC-Events-Sample web application gets events statistics that are gathered by a server-side application that has to be deployed on the server where IBM Infosphere INformation Server is running. To deploy the server-side application, follow the instructions below:
  1. Download following jar file: GC-Events-Sample-1.0.jar
  2. Copy the downloaded file to the server where IBM Infosphere Information Server is running,
  3. Execute following command: java -jar GC-Events-Sample-1.0.jar localhost http://gc-events-sample.mybluemix.net demo
The last parameter "demo" produces demo events, for example in the case that the Infosphere Information Server does not produce events. It can be omitted.
